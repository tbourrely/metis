# Metis

> Metis was the pre-Olympian goddess of wisdom, counsel and deep thought, source: [Wikipedia](https://en.wikipedia.org/wiki/Metis_(mythology))

Metis is a web application designed to facilitate technology watch and keeping up with the latest trends in various fields.

## Goals

* Aggregate articles, news, and updates from multiple sources.
* Provide a user-friendly interface for browsing and reading content.
* Simple to use with minimal setup required, with self-hosting in mind.

## Tech Stack

* Frontend: React, Tailwind CSS
* Backend: Node.js, NestJS, TypeScript
* Database: PostgreSQL

## Status

WIP - Under active development

## Setup

1. Clone the repository.

2. Create a `.env` file in the root directory and configure the necessary environment variables (see __.env.tpl__).

3. Start the application using Docker Compose:

```bash
docker-compose up -d
```

## Inspirations

* [pocket](https://getpocket.com/): For saving and organizing articles.
* [feedly](https://feedly.com/): For aggregating news and articles from multiple sources.
* [wallabag](https://www.wallabag.org/): For read-it-later functionality.
